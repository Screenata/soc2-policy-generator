# ISO 27001:2022 Annex A â€” Control Mappings

This file maps each policy to its relevant ISO 27001:2022 Annex A controls. Used by the agent to populate the `satisfies.ISO27001` field in policy YAML frontmatter when ISO 27001 is selected.

## How to Use

When generating a policy for a user who selected ISO 27001:
1. Look up the policy ID in the table below
2. Include all listed Annex A codes in the policy's `satisfies.ISO27001` frontmatter
3. Reference the control descriptions in the policy body where appropriate
